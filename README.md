This is a set of scripts and utilities to drive the lighting on the arches and plinths in Blue Paradox Room 3. This replaces (or is a band-aid for) an existing touchdesigner patch program.

## Scripts

### capture.py

Grabs sacn data off the network for the specified universes. Stores the first packet seen in each universe. Reports if not all the expected universes were captured. Saves the results to a json file with the current date and time.

```sh
usage: Capture sacn at a point in time [-h] -u UNIVERSES [-t TIME]

options:
  -h, --help            show this help message and exit
  -u UNIVERSES, --universes UNIVERSES
                        a comma-separated list of universes to capture
  -t TIME, --time TIME  How long to record data for (seconds)
```

### send.py

Sends sacn data over the network to the specified universes. Can transmit the DMX data from a file generated by `capture.py` with the `--file` flag. Or can send debug data of a specific color (i.e. ``--debug --color green`).

This is the script that is run automatically on startup, via the batch file at `start_show.bat`.

```sh
usage: Send sacn lighting data [-h] [--debug] [--color {red,green,blue}] [-u UNIVERSES]
                               [-f FILE]

options:
  -h, --help            show this help message and exit
  --debug               Send a fixed blue value to all universes
  --color {red,green,blue}
                        debug color (only valid with --debug)
  -u UNIVERSES, --universes UNIVERSES
                        a comma- or dash-separated list of universes to capture (e.g. 1,2,3-5)    
  -f FILE, --file FILE  json data file to open
```

Arches use DMX universes 47-> 160
